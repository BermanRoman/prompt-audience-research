# Глубинное исследование целевой аудитории

Мастер-промт проводит тебя через полное исследование целевой аудитории и собирает два готовых продукта: структурированную базу знаний о клиенте мечты и продающий лендинг.

## Для кого создан этот мастер-промт? 

Для маркетологов, экспертов и предпринимателей, которым нужно понять свою аудиторию и упаковать под них предложение.

## Как пользоваться?

1. Создай новый проект в Claude.
2. Нажми «Copy» на блоке с промтом ниже и вставь его в поле «Инструкции» проекта.
3. Напиши боту любое сообщение — он сам поздоровается и начнёт исследование, задавая по одному вопросу за раз.
